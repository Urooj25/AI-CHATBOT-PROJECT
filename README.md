# AI-CHATBOT-PROJECT


Welcome to my **AI Chatbot Project**! This project is built using Python and Hugging Face models to create a smart, interactive, and conversational chatbot. Perfect for personal assistants, websites, or learning AI integration.

---

## 🛠️ Features

- Conversational AI with context awareness  
- Easy to customize responses  
- Integration with Hugging Face models  
- Secure setup using `.env` for API keys  
- Lightweight and fast for local or cloud deployment
